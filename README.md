# 🐱 DevKitty

**Track the most profitable PumpFun developers** - analyze their launches, profits, success rates, and creator fees with our purr-fect analytics platform.

## 🚀 Live Demo

- **Production**: https://devkitty-bbtqrp8gn-senzsosm-gmailcoms-projects.vercel.app
- **GitHub**: https://github.com/Hashdevlol/devkitty

## ✨ Features

- **Developer Leaderboard**: Real-time ranking of top PumpFun developers by total profit
- **Detailed Analytics**: Track success rates, launch counts, creator fees, and trading volume
- **Individual Profiles**: Deep-dive into each developer's launch history and performance
- **Responsive Design**: Beautiful dark theme with gradient animations
- **Real-time Data**: Powered by Helius API for accurate Solana blockchain data

## 📊 What We Track

- **Total Profits**: Comprehensive P&L across all token launches
- **Success Rates**: Percentage of profitable vs. unprofitable launches
- **Creator Fees**: Revenue from initial token sales and trading fees
- **Launch Volume**: Total trading volume generated by developer tokens
- **Performance Streaks**: Current and best winning streaks
- **Token Analytics**: Individual token performance and multipliers

## 🛠️ Tech Stack

- **Frontend**: Next.js 16, React 19, TypeScript
- **Styling**: Tailwind CSS with custom gradients
- **Data**: Helius API for Solana blockchain analytics
- **Deployment**: Vercel
- **UI Components**: Custom components with shadcn/ui styling

## 🔧 Setup & Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Hashdevlol/devkitty.git
   cd devkitty
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   ```bash
   cp .env.local.example .env.local
   ```
   
   Add your Helius API key:
   ```
   HELIUS_API_KEY=your_helius_api_key_here
   ```

4. **Run development server**:
   ```bash
   npm run dev
   ```

5. **Open**: http://localhost:3000
